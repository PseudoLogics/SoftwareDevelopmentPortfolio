Inventory Management Application

CS 499 Capstone • Software Engineering • Algorithms • Databases

Capstone Artifact

This project represents my final capstone artifact for the Computer Science program. The application began as a basic inventory system and was enhanced to demonstrate growth across software engineering, algorithms, and database design. The final version supports item management, category organization, and real-time search functionality while reflecting improvements in architecture, scalability, and overall user experience.

Software Design & Engineering

I improved the architecture of the application by separating concerns between the user interface, business logic, and database interactions. The RecyclerView adapter was refactored to remove direct database operations, improving maintainability and scalability. I also implemented stronger input validation and error handling, which increased reliability and created a more polished user experience. These changes demonstrate my ability to apply professional software engineering practices to an existing system.

Algorithms & Data Structures

This enhancement focused on improving how data is processed and retrieved within the application. I refined search and filtering functionality to provide more efficient item lookup and smoother real-time interaction. By improving the way data flows through the system, the application now performs better as the dataset grows. This demonstrates my understanding of how algorithmic efficiency and data structure design impact performance and usability.

Database Design

The database layer was redesigned using normalization principles to separate items and categories into structured relational tables. Indexes were added to improve query performance, and new attributes such as location and barcode support were introduced. These changes improved data integrity, reduced redundancy, and made the application more scalable. This enhancement highlights my ability to design and implement efficient, real-world database solutions.
